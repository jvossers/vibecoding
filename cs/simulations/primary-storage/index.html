Key Facts

RAM (Random Access Memory) is volatile — it loses all data when power is turned off. It stores currently running programs and data.
ROM (Read Only Memory) is non-volatile — it retains data without power. It stores the boot-up instructions (BIOS/firmware).
Virtual memory uses a section of the hard drive as if it were extra RAM. Pages of data are swapped between RAM and disk.
Thrashing occurs when the system spends more time swapping pages than executing programs — caused by too little RAM.
Exam tip: When asked "explain why adding more RAM speeds up a computer", say: more programs can be held in RAM simultaneously, reducing virtual memory swapping, which means less thrashing and faster access times.
