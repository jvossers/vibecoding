Key Facts

Lossless compression reduces file size without losing any data. The original can be perfectly reconstructed. Example: Run-Length Encoding (RLE).
Lossy compression permanently removes some data to achieve smaller file sizes. Quality degrades but the file is much smaller. Example: JPEG.
RLE replaces runs of the same character with a count and the character. "AAABBB" becomes "3A3B".
Exam tip: You must explain the difference between lossy and lossless, give examples, and state when each is appropriate (lossless for text/programs, lossy for images/audio).
