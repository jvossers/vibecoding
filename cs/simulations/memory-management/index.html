Key Facts

RAM (Random Access Memory) is volatile primary storage. It holds data and instructions currently in use by the CPU.
Virtual memory uses a section of the hard drive as if it were RAM. When RAM is full, pages are swapped to disk.
Paging divides memory into fixed-size blocks (pages). It avoids the problem of external fragmentation.
Contiguous allocation requires a process to occupy a single continuous block of memory. It can lead to external fragmentation.
Exam tip: Be able to explain why virtual memory can slow down the computer: constant swapping between RAM and disk (thrashing) is much slower than accessing RAM directly.
