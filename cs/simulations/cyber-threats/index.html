Key Facts

Malware is malicious software designed to damage, disrupt, or gain unauthorised access. Types: virus, worm, trojan, spyware, ransomware.
Social engineering manipulates people into revealing confidential information. Phishing uses fake emails/websites.
SQL injection inserts malicious SQL code into input fields to access or modify a database.
Brute force tries every possible combination until the correct password is found.
DoS/DDoS floods a server with requests to make it unavailable. DDoS uses multiple compromised machines (botnet).
Exam tip: For any threat question, state: (1) what it is, (2) how it attacks, (3) what damage it causes.
