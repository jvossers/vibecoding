Key Facts

Integer — a whole number (positive, negative, or zero). Used for counting and calculations without decimals.
Real (Float) — a number with a decimal point. Used when precision matters (prices, measurements).
Boolean — True or False only. Used for conditions and flags.
Character — a single letter, digit, or symbol. Stored as an ASCII code (0-127).
String — a sequence of characters. Even numbers can be strings ('07700') — no arithmetic possible.
Casting converts a value from one data type to another. Not all casts are valid (e.g., int('Hello') fails).
Exam tip: Phone number → String (leading zeros, no maths). Age → Integer. Price → Real. Yes/No → Boolean. Always explain WHY.
