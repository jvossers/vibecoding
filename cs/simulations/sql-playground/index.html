Key Facts

SQL (Structured Query Language) is used to search, sort and manage data in relational databases.
SELECT specifies which columns to return. Use * for all columns.
WHERE filters rows based on conditions. You can combine conditions with AND/OR.
ORDER BY sorts results. ASC = ascending (default), DESC = descending.
Exam tip: In exams, SQL keywords are not case-sensitive, but write them in CAPITALS for clarity: SELECT, FROM, WHERE, ORDER BY, AND, OR.
