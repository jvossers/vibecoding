Key Facts

Binary (base 2) uses only 0 and 1. Each column doubles from right to left: 1, 2, 4, 8, 16, 32, 64, 128.
Hexadecimal (base 16) uses 0–9 then A–F. One hex digit = exactly 4 binary bits.
Exam tip: To convert binary → hex, split into nibbles (groups of 4 bits) and convert each group separately.
