Key Facts

Sample rate is the number of audio samples captured per second, measured in Hertz (Hz). CD quality = 44,100 Hz.
Bit depth is the number of bits used to record each sample. Higher bit depth = more precise amplitude values.
Nyquist theorem: the sample rate must be at least double the highest frequency to accurately reproduce sound.
Exam tip: File size = sample rate × bit depth × duration × channels ÷ 8 bytes. Always show this formula in exams.
