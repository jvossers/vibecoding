Key Facts

An embedded system is a computer built into a larger device with a dedicated function. It is designed for one specific task.
A general-purpose system (like a PC or smartphone) can run many different programs and perform a variety of tasks.
Embedded systems are typically low-power, low-cost, and have limited storage. They usually don't have a standard OS.
Exam tip: Check: Is it built into a larger device? Does it have a dedicated function? Can you install new software on it? If built-in + dedicated + no new software = embedded.
