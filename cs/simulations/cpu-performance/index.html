Key Facts

Clock Speed (GHz) determines how many instructions the CPU can execute per second. Higher clock speed = faster processing.
More Cores allow the CPU to process multiple tasks simultaneously (parallel processing). But not all tasks can be split across cores.
Cache is small, fast memory on the CPU chip. Frequently used data is stored here to avoid slower RAM access. Larger cache = fewer delays.
Exam tip: When asked "explain how X affects CPU performance", always link the factor to processing speed and give a specific example.
