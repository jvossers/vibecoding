Key Facts

Fragmentation occurs when files are split into pieces stored in non-contiguous locations on a hard drive.
Defragmentation reorganises the data so each file's blocks are stored together. This reduces read/write head movement and speeds up access.
SSDs do not need defragmentation because they have no moving parts — any memory cell can be accessed equally fast (no seek time).
Exam tip: In exams, explain that defragmentation improves HDD performance by reducing the distance the read/write head must travel, decreasing access time.
