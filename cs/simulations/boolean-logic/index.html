Add gate:

Click a gate button then click the canvas to place it. Drag between connection points to wire. Click an Input node to toggle 0/1.

Key Facts

AND gate: Output is 1 only when ALL inputs are 1.
OR gate: Output is 1 when ANY input is 1.
NOT gate: Inverts the input — 0 becomes 1, 1 becomes 0.
Exam tip: You must be able to complete truth tables for AND, OR and NOT gates, and combinations of gates.
