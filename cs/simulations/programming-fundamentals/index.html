Key Facts

Variable — stores a value that can change during the program. A constant stores a fixed value that cannot be changed once set.
Arithmetic operators: + (add), - (subtract), * (multiply), / (divide), DIV (integer division), MOD (remainder).
Selection (IF/ELSE) lets the program choose different paths based on a condition. Iteration (FOR/WHILE) lets the program repeat code.
Exam tip: In pseudocode: declare variables before use, use meaningful names, indent code inside IF/FOR/WHILE blocks.
