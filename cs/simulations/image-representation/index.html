Key Facts

File size = width × height × colour depth (in bits) ÷ 8 to get bytes. This is the uncompressed, metadata-free size.
Colour depth is the number of bits used per pixel. 1-bit = 2 colours, 8-bit = 256 colours, 24-bit = ~16.7 million colours.
Resolution is the number of pixels in the image. Higher resolution = more detail but larger file size.
Exam tip: You must be able to calculate file sizes. Remember: file size (bits) = width × height × colour depth.
