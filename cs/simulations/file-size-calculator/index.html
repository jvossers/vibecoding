Key Facts

Unit conversions: 1 byte = 8 bits. 1 KB = 1,000 bytes. 1 MB = 1,000,000 bytes. 1 GB = 1,000,000,000 bytes.
Colour depth is the number of bits used to represent each pixel. More bits = more colours but larger files.
Sample rate is the number of samples taken per second. Higher sample rates capture more detail in the sound.
Exam tip: Always show your working in file size calculations. Write the formula first, substitute values, then convert units step by step.
